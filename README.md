# 🐍 Python Power User Course
## For CBSE 7th Grade Students

Welcome to your Python learning journey! This course is designed specially for you — a 13-year-old student who loves cricket and gaming. By the end of this 12-week course, you'll be able to:

- ✅ Create amazing drawings and animations with Python
- ✅ Build smart calculators and report carīds
- ✅ Analyze cricket statistics like a pro
- ✅ Work with real databases (like the pros use!)
- ✅ Fetch live data from the internet
- ✅ Prepare yourself for powerful automation tools like n8n and SQL

---

## 📁 Course Structure

### Phase 1: Drawing with Python (Weeks 1-2)
Start your journey by making Python draw for you! Create shapes, patterns, and your own art generator.

| File | What You'll Learn |
|------|-------------------|
| `01_hello_variables.py` | Print messages, store player stats in variables |
| `02_turtle_basics.py` | Make Python draw a cricket pitch |
| `03_for_loops_shapes.py` | Draw triangles, squares, pentagons, hexagons |
| `04_colors_parameters.py` | Add colors and create IPL team logos |
| `project_turtle_art_generator.py` | 🏆 Build your own mandala art generator! |

### Phase 2: Smart Logic (Weeks 3-4)
Teach Python to make decisions and think logically.

| File | What You'll Learn |
|------|-------------------|
| `05_if_else_conditions.py` | Rate cricket performances |
| `06_input_typecasting.py` | Calculate batting averages correctly |
| `07_while_loops.py` | Simulate a cricket innings |
| `08_boolean_logic.py` | Check playoff qualifications with AND/OR |
| `project_smart_report_card.py` | 🏆 Build a complete report card system! |

### Phase 3: Lists & Collections (Weeks 5-6)
Manage multiple items like player squads and scoreboards.

| File | What You'll Learn |
|------|-------------------|
| `09_lists_indexing.py` | Access top scorers from a list |
| `10_list_methods_sorting.py` | Sort teams, find highest/lowest scores |
| `11_dictionaries.py` | Create player profile cards |
| `12_loops_on_lists.py` | Filter players by criteria |
| `project_class_leaderboard.py` | 🏆 Build a gaming leaderboard! |

### Phase 4: Functions (Weeks 7-8)
Write reusable code like a professional programmer.

| File | What You'll Learn |
|------|-------------------|
| `13_defining_functions.py` | Stop repeating yourself with functions |
| `14_parameters_return.py` | Create cricket scoring calculators |
| `15_functions_calling_functions.py` | Build complex scenes from simple parts |
| `16_scope.py` | Debug tricky variable errors |
| `project_turtle_library.py` | 🏆 Create your own drawing library! |

### Phase 5: Data & Databases (Weeks 9-10)
Work with real data files and databases — the skills used in every app!

| File | What You'll Learn |
|------|-------------------|
| `17_reading_csv.py` | Load IPL player stats from files |

| `20_dynamic_queries.py` | Build search tools with user input |
| `project_cricket_stats_db.py` | 🏆 Complete cricket stats app! |

### Phase 6: Automation & APIs (Weeks 11-12)
Connect to the internet and build data pipelines!

| File | What You'll Learn |

## 🚀 How to Use This Course

1. **Open each file in Thonny or Replit** — Start with `01_hello_variables.py`
2. **Read ALL the comments** — They explain everything!
3. **Run the code** — See what happens on screen
4. **Complete the "🔥 TRY THIS" challenges** — They're at the end of every file
5. **Don't skip files** — Each one builds on the previous

---

## 🎯 Special Features in Every File

Look for these special comments:

- `# 🧠 LOGIC SEED:` — Hidden lessons about algorithms and data structures
- `# 🔗 FUTURE TOOL:` — How this connects to SQL, n8n, and other tools
- `# 🔥 TRY THIS:` — Your challenges at the end of each file
